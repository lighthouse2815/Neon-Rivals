# Asset Guide

## Style

- Original neon cyber arena with dark graphite floor, cyan and magenta emissive accents, and sharp silhouettes.
- HUD uses clean competitive shapes, restrained glow, and strong contrast.

## Runtime assets

- Players, projectiles, particles, arena tiles, and UI chrome are generated or drawn from original vectors and procedural textures.
- Sound effects are synthesized from original wave data and bundled locally.

## Organization

- Source-oriented asset helpers live under `apps/client/src/assets`.
- Evidence screenshots live under `docs/evidence/`.
- Any future exported source art should be tracked alongside the runtime asset it feeds.
